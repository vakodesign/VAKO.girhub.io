Desarrollo Interactiva & AR

Arquitectura de Experiencias WebAR para Eventos Globales.

Implementación de flujos de acceso mediante códigos QR para el despliegue de experiencias de Realidad Augmented basadas en web (WebAR). El enfoque principal fue la optimización de activos 3D para asegurar tiempos de carga mínimos y compatibilidad multiplataforma sin necesidad de aplicaciones externas, maximizando la tasa de interacción durante el festival.

Digitalización de Productos Mediante Interfaces Interactivas 3D.

Desarrollo de un sistema de menú digital interactivo que permite la visualización fotorrealista de productos en el espacio físico del usuario. Se diseñó la interfaz de usuario (UI) y la lógica de interacción para facilitar la navegación y mejorar la experiencia de compra mediante la integración de modelos 3D con materiales PBR optimizados para navegadores móviles.
